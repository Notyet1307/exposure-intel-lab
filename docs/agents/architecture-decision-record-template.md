# ADR Template

Create ADRs in `docs/adr/` using sequential numbers:

```text
docs/adr/0001-short-slug.md
```

## Template

```md
# <Short title>

Status: proposed | accepted | deprecated | superseded by ADR-XXXX
Date: YYYY-MM-DD

## Context

<1-3 paragraphs explaining the forces and constraints.>

## Decision

<The decision.>

## Consequences

<Tradeoffs, operational impact, testing impact, migration impact.>

## Alternatives considered

<Only include meaningful alternatives future engineers or agents may re-suggest.>
```
